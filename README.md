# Korg Logue Reverse Echo Delay FX

>Echos reversed audio input

## Functionality

|Knob|Function|
|-|-|
|A|Sets the length (1 - 2 seconds) of the audio to reverse|
|B|Sets trailing echo length|
|Shift B|Wet/Dry Mix|
